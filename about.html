About AuthentiCite

AuthentiCite is an intelligent reference resolver and verifier. It converts plain-text citations in any format into clean BibTeX entries — and flags references that look suspicious or AI-generated before they make it into your manuscript.

Version 4 introduces multi-strategy parallel API search, semantic abbreviation expansion, and a dedicated AI hallucination detector that catches fabricated papers at the title level.

🔍

Smart Preprocessing

Strips rich-text artifacts, detects DOIs and arXiv IDs for fast-path lookup before falling back to title search.

⚡

Parallel API Search

Queries Crossref, Semantic Scholar, and OpenAlex simultaneously — picking the best match across all three.

🤖

Hallucination Detection

Flags AI-generated references using title template pattern analysis and known predatory journal signals.

📊

Verified vs. Flagged

Output is split into two BibTeX sections — clean verified entries and suspicious flagged ones — so you always know what to trust.

🔤

Abbreviation Expansion

Understands that "UAV" = "unmanned aerial vehicle" for better fuzzy title matching.

🛡️

LLM Structural Check

References not found in any database get a secondary structural plausibility check via AI validation.

APIs Used

AuthentiCite uses three free, open academic APIs — no API key required. All are free to use for research purposes.

Crossref Semantic Scholar OpenAlex Anthropic Claude (LLM validator)

Supported Citation Formats

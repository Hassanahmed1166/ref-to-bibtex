About ref→bib v4

    
An intelligent reference resolver with multi-strategy search and AI hallucination detection.

    
Improvements in v4:

    
    
      
  • Smart preprocessing: strips rich-text artifacts, detects DOIs/arXiv IDs for fast-path lookup
    • 
      
  • Multi-strategy search: tries parsed title → raw string → author+year combinations
    • 
      
  • Semantic abbreviation expansion: "uav" ↔ "unmanned aerial vehicles", "nlp" ↔ "natural language processing", etc.
    • 
      
  • Adaptive similarity: lower threshold (0.55) for abbreviated/partial titles that expand well
    • 
      
  • Parallel API calls (Crossref + Semantic Scholar + OpenAlex simultaneously)
    • 
      
  • LLM structural validator for references not found in any API
    • 
      
  • AI hallucination detector: title template patterns + fake journal signals
    • 
    

    
APIs used: Crossref, Semantic Scholar, OpenAlex (all free, no key needed).
